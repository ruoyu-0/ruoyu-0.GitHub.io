无题

发表于2001-03-16|更新于2023-03-16

今夜听了 生命因你而火热 忍不住在听歌的时候写点什么东西。我忍不住想和别人交流自己难以抑制的感情，却又畏首畏尾，无法表达。听歌，学习历史，写日记，我希望这代替毫无意义的闲聊，充实我的生活。我感觉我在慢慢学会蜷成一团，慢慢成为一个茧，慢慢像我之前想要的那样，在安静中慢慢萌发。
# 在茧里，我要学会原谅。
原谅我经历的这些，原谅自己，原谅别人。人非圣贤，孰能无过？我回望着我经历的一切，细数自己没有完成的承诺，辜负的期望。但有很多很多目标，理由，想做的事。还有很多很多人的盼望，很多人的关怀，很多人的帮助，很多人的安慰。原谅自己吧。
We ant done nothing wrong but we ant done noting right
我学了医，是为了在需要我帮助的时候不再感到无能为力。大学五年，我参与了相声队，认识了很多有意思的人，让很多人都认可我。也曾成为小舞台的焦点，也曾成为很多人眼里的光。选择导师，我选到了我希望选到的老师，认识了我认可的小导，接触了气氛很好的师兄师姐。在大学里，我谈了一段恋爱，我认真地去喜欢过了，认真去思考过了。面对这些，我没有屈从于对孤单的害怕，而是坚持了自己，感情来源于喜欢。面对这些，我没有屈从于欲望，我认识到了自己的认真，负责，执着，勇敢，肩负责任，没有逃避。面对困难，我知道我会后悔我会难过，但在最后我也没有纠缠，而是果断放手。我爱我爱的人，我认真无私地爱着，干净轻松地离开，没有狗血，没有纷纷扰扰。我相信哪怕再想起我，再提起我，也能说这是一次不会后悔的感情。我的喜欢在表达的时候，真诚而热烈。

俏皮一点，缺点明天再说。嘻嘻。
